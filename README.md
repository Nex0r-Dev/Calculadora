# Calculadora

Una calculadora avanzada en Python con interfaz gráfica utilizando Tkinter. Incluye operaciones básicas, avanzadas y funciones trigonométricas, además de un historial de cálculos.

## Características

- **Operaciones básicas**: Suma, resta, multiplicación, división.
- **Operaciones avanzadas**: Potencia, raíz cuadrada.
- **Funciones trigonométricas**: Seno, coseno, tangente (en grados).
- **Interfaz gráfica**: Fácil de usar con botones coloreados y diseño intuitivo.
- **Historial**: Registra todas las operaciones realizadas, con opción de ver y limpiar.
- **Manejo de errores**: Validación de entradas y mensajes de error.

## Requisitos

- Python 3.x (incluye Tkinter por defecto en la mayoría de instalaciones).

## Instalación

1. Clona este repositorio:
   ```bash
   git clone https://github.com/tu-usuario/calculadora.git
   cd calculadora
   ```

2. Ejecuta el script:
   ```bash
   python calculadora_pro.py
   ```

## Uso

- **Entrada de números**: Usa los botones numéricos o escribe directamente en el campo de entrada.
- **Operaciones binarias**: Ingresa la expresión (ej. `5+3`) y presiona `=`.
- **Funciones**: Ingresa el valor y presiona el botón de función correspondiente (ej. `30` luego `sin`).
- **Historial**: Presiona `Hist` para ver el historial en una ventana emergente, o `Clear Hist` para limpiarlo.
- **Limpiar**: Presiona `C` para borrar la entrada actual.

## Estructura del proyecto

- `calculadora_pro.py`: Archivo principal con la lógica de la calculadora y la interfaz gráfica.
